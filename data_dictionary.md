| Variable                             | Description                                                              |
|:-------------------------------------|:-------------------------------------------------------------------------|
| participant_code                     | Unique identifier for each participant.                                  |
| consent                              | Indicator of participant consent (1 = consent given).                    |
| generation                           | Participant's generation within a cultural transmission chain (1–4).     |
| session_code                         | Unique session ID generated by the platform (e.g., oTree or Prolific).   |
| treatment_appeal                     | Experimental treatment condition: 'high_appeal' or 'low_appeal'.         |
| chain_code                           | Unique identifier for each cultural transmission chain.                  |
| selected_feedback_index              | Index (1–3) of the selected cultural demonstrator.                       |
| selected_feedback_message            | Advice message chosen from the previous generation.                      |
| selected_feedback_grid_state         | Grid configuration used by the chosen demonstrator.                      |
| selected_feedback_author             | Participant code of the selected demonstrator.                           |
| selected_feedback_score              | Final trial score of the selected demonstrator.                          |
| feedback_message                     | Advice written by the participant for the next generation.               |
| feedback_grid_state                  | Configuration of the participant's solution in the final trial.          |
| feedback_score                       | (Realized) Score of the configuration in the final trial.                |
| birthyear                            | Participant's birth year.                                                |
| gender                               | Self-reported gender of the participant.                                 |
| period                               | Trial number within each participant (1 to 6).                           |
| revenue                              | Raw output of the field before subtracting cost.                         |
| cost                                 | Total cost of nutrients used in the trial.                               |
| points                               | Net score: revenue minus cost.                                           |
| plants_treated                       | Number of plants that received at least one nutrient.                    |
| grid_state                           | Full configuration of nutrient allocation                                |
| period_in_chain                      | Trial number indexed across participants in the chain.                   |
| num_plants_fed                       | Number of plants fed with any nutrient in the trial.                     |
| selected_feedback_grid_state_flatten | Flattened format of selected feedback grid state.                        |
| grid_state_flatten                   | Flattened string representation of participant's solution in the trial.  |
| feedback_grid_state_flatten          | Flattened string representation of feedback grid state.                  |
