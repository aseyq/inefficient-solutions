# The Emergence and Cultural Persistence of Ineffective Solutions

This repository contains the code and data necessary to reproduce analyses and figures in the paper _"The Emergence and Cultural Persistence of Ineffective Solutions."_ All analyses are conducted using R.

## Repository Structure
| Path | Type | Description |
|---|---|---|
| `run_all.R` | R script | Runs all R scripts in `code/` and writes outputs to `figures/`. |
| `requirements.txt` | Python deps | Python dependencies for semantic processing and GPT classification scripts. |
| `code/` | Directory | Analysis scripts for main and supplementary figures/tables. |
| `code/fig2.R`, `code/fig3.R`, `code/fig4.R`, `code/fig5_data.R` | R scripts | Main figure generation and supporting data prep. |
| `code/figA1.R` ... `code/figA10.R` | R scripts | Supplementary figure scripts. |
| `code/tableA1.R`, `code/tableA2.R`, `code/tableA4.R` | R scripts | Supplementary table scripts. |
| `code/figA7_calclulate_semantic.py` | Python script | Creates semantic similarity inputs in `data/processed/` for `code/figA7.R`. |
| `code/classify_gpt5.py` | Python script | Classifies advice into categories using GPT (requires `OPENAI_API_KEY`). |
| `code/compare_manual_and_gpt_coding.R` | R script | Compares manual coding with GPT coding outputs. |
| `data/` | Directory | Raw and derived datasets used by analyses. |
| `data/df_long.csv` | CSV | Main dataset used across analyses. |
| `data/df_advice_manual_coding.csv` | CSV | Manual coding labels for advice messages. |
| `data/df_advice_gpt5.csv` | CSV | GPT-based coding output for advice messages. |
| `data/gpt_template.csv` | CSV | Input template used by `code/classify_gpt5.py`. |
| `data/processed/` | Directory | Processed outputs (including semantic similarity files). |
| `figures/` | Directory | Output figures generated by scripts. |
| `data_dictionary.md` | Documentation | Variable descriptions for the datasets. |

## Requirements

- R packages (installed in current environment):
   - `tidyverse==2.0.0`
   - `lme4==1.1.36`
   - `lmerTest==3.1.3`
   - `sjPlot==2.9.0`
   - `purrr==1.0.2`

- Python packages (from `requirements.txt`):
   - `numpy==2.4.2`
   - `openai==2.21.0`
   - `pandas==3.0.1`
   - `sentence-transformers==5.2.3`


You can install the required packages with the command:

```r
install.packages(c("tidyverse", "lme4", "lmerTest", "sjPlot", "purrr"))
```

Install Python dependencies with:

```bash
pip install -r requirements.txt
```

Notes for the semantic script:
- `code/figA7_calclulate_semantic.py` uses the public model `all-MiniLM-L6-v2`.
- No Hugging Face account is required.
- Requires download of the model the first time, which may take a few minutes.

## Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/aseyq/inefficient-solutions.git
   ```

2. **Install R packages**:

   ```r
   install.packages(c("tidyverse", "lme4", "lmerTest", "sjPlot", "purrr"))
   ```

3. **Install Python dependencies** (for semantic preprocessing used by Fig A7):

   ```bash
   pip install -r requirements.txt
   ```

4. **Run semantic preprocessing for Figure A7**:

   ```bash
   python code/figA7_calclulate_semantic.py
   ```

   This creates files in `data/processed/` that are required by `code/figA7.R`.

5. **Run analyses and generate figures**:

   Run all R scripts in `code/`:

   ```bash
   Rscript run_all.R
   ```

   Or run specific scripts manually:

   ```r
   source("code/fig2.R")
   source("code/fig3.R")
   source("code/fig4.R")
   source("code/figA7.R")
   ```

6. Output files (figures, tables, and intermediate products) are written to `figures/`.

## Data

The experiment dataset is `data/df_long.csv`. Please refer to the [data dictionary](data_dictionary.md) for detailed descriptions of each variable.
